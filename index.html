Conceitos básicos de Git

  1. Primeiro faça um git init
  2. Segundo adicione com o git add os arquivos que deseja (Você pode usar um git status para verificar se os arquivos foram adicionados)
  3. Terceiro faça o primeiro commit na sua maquina com o git commit -m "nome do commit"
  4. Quarto você pode alterar o nome de master para algum outro com o comando git branch -M "main", normalmente se usa main ou master
  5. Quinto, para conseguir mandar para o GitHub faça o comando git remote add origin link do repositorio no GitHub
  6. Sexto, você só vai conseguir visualizar os arquivos no GitHub após dar um push com o comando git push -u origin main,
